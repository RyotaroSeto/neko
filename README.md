```
    `""""""""""""`
```


## インストール
```
$ brew install RyotaroSeto/tap/neko
```

## 最新のversionに更新
```
$ brew upgrade RyotaroSeto/tap/neko
```

## 使い方
### 猫を呼び出すと猫が現れます。
```
$ neko
            /\___/\
           ( o   o )
           (  =^=  )
           (        )
           (         )
        ___v__^__v___
       /             \
      |               |
      |               |
       \             /
        `""""""""""""`
```

### 猫の名前を提案してくれます。
```
$ neko name
Ryotaro
```

### 名前を呼んだら応答してくれます。
```
$ neko Ryotaro
            /\___/\
           ( o   o )
           (  =^=  )   ニャー
           (        )
           (         )
        ___v__^__v___
       /             \
      |               |
      |               |
       \             /
        `""""""""""""`
```
